### **DSPy 是什么？**

DSPy 是一个**声明式框架**，用于构建模块化的人工智能软件。其核心理念是 **“编程，而非提示”**，旨在将开发者的焦点从繁琐的字符串提示工程转移到结构化、可维护的代码编程上。

### **核心特性与优势**

1. **模块化编程**：允许开发者使用**自然语言模块**来描述AI组件的行为（通过“签名”定义输入/输出），而非直接操作脆弱的提示字符串。这使得AI系统更**可靠、可维护、可移植**。
2. **自动优化**：提供多种**优化器**，能够自动将高级程序代码编译成针对特定语言模型的高效提示或权重更新。优化方式包括：
   * 合成优质的少样本示例。
   * 探索更好的自然语言指令。
   * 为微调生成数据集。
3. **开源生态**：源于斯坦福NLP的研究，拥有活跃的社区和丰富的生态系统，持续在优化器、程序架构和实际应用方面取得进展。

### **主要功能**

* **快速构建**：提供如 `dspy.Predict`、`dspy.ChainOfThought`、`dspy.ReAct` 等预置模块，可轻松构建分类器、RAG管道、智能体等。
* **支持多种模型**：兼容 OpenAI、Anthropic、Gemini、本地模型等多种语言模型提供商。
* **低成本优化**：典型的优化运行成本低廉（约2美元），耗时短（约20分钟），能显著提升模型在特定任务上的表现（示例中准确率从24%提升至51%）。

### **总结**

DSPy 通过将AI系统**抽象为可组合、可优化的代码模块**，极大地简化了复杂AI应用的开发流程。它降低了提示工程的难度，提高了系统的迭代速度和性能上限，是构建下一代可靠、可维护AI软件的重要工具。
